    # ExpandableText 可展开文本

    # 基础用法

    基础的多行可展开文本用法，支持IE10+、Safari、Firefox、Chrome、Edge。



    浮动元素是如何定位的
    正如我们前面提到的那样，当一个元素浮动之后，当一个元素浮动之后，它会被移出正常的文档流，然后向左或者向右移，一直平移直到碰到了所处的容器的边框，或者碰到另外一个浮动的元素。在下面的图片中，有三个红色的正方形。其中有两个向左浮动，一个向右浮动。要注意到第二个向左浮动的正方形被放在第一个向左浮动的正方形的右边。如果还有更多的正方形这样浮动，它们会继续向右堆放，直到填满容器一整行，之后换行至下一行。

    可以使用text属性来定义 ExpandableText 的文本，同时也支持使用默认插槽传递文本数据，默认展示3行。

  <div class="expandable-text-demo">
    <w-expandable-text
      :text="text"
    ></w-expandable-text>
    <br>
    <w-divider />
    <br>

    # 属性

    参数 说明 类型 是否必须 可选值 默认值
    text 文本数据 string 否 — —
    maxLines 展示文本行数，目前支持最大10行 string / number 否 — 3
    expandText 展开按钮文本 string 否 — '展开'
    collapseText 收起按钮文本 string 否 — '收起'
