### 属性

| 参数   | 说明                                    | 类型           | 是否必填 | 可选值 | 默认值 |
| ------ | --------------------------------------- | ---------------| ------ | ------ | ------ |
| color   | 滚动条滑块颜色                                | string          | 否 | —      | #cecece      |
| width   | 滑块宽度,仅垂直生效                                    | string  | 否 |  —      | 6px     |
| right  | 滚动条右边距,仅垂直生效                                | string          | 否 | —      | 2px      |
| height  | 滑块高度,仅水平生效                                | string          | 否 | —       | 6px      |
| bottom  | 滚动条下边距,仅水平生效                                | string           | 否 | —      | 2px      |
| noResize | 尺寸不变时可以优化性能                              | boolean          | 否 | —      | false      |
